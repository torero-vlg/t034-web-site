Сборка:
- запустить Publish на проекте T034. Сайт соберётся в папку publish
- в папку publish скопировать папки Content, font, Scripts, Upload

Первоначальная установка:
- копируем сайт на сервер
- копируем пустую БД
- заходим на /Setting и заполняем почту админа и код приложения на яндексе, почта тоже должна быть на яндексе
- нажимаем сохранить, после этого в БД будет создан пользователь с правами админа и настройки OAuth
- авторизовываемся по адресу /Account/Logon
